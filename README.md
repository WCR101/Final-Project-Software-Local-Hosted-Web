Athenaeum is a full-stack, Dockerized library management system designed to simulate a real-world application with a clean and modular architecture. The project combines a PostgreSQL database (Supabase-compatible), a Node.js and Express backend for handling business logic, a PostgREST service for automatic API generation, and a lightweight frontend served through Nginx. All components are orchestrated using Docker Compose, allowing the entire system to be spun up quickly and consistently across environments. The application supports core library operations such as managing books, patrons, and loans, along with features like overdue tracking, fine calculation, and system statistics. Business rules are enforced at both the API and database levels, including limits on the number of books a patron can check out, automatic fine accumulation for overdue items, and restrictions when outstanding balances exist. This project demonstrates practical experience in full-stack development, REST API design, relational database modeling, and containerized deployment, while emphasizing scalability, separation of concerns, and real-world system behavior.
